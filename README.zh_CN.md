## 安装

```bash
# pnpm
pnpm add @tomjs/vite-plugin-monaco-editor -D

# yarn
yarn add @tomjs/vite-plugin-monaco-editor -D

# npm
npm i @tomjs/vite-plugin-monaco-editor --save-dev
```
